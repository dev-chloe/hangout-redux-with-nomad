# Vanilla-Redux and React-Redux

Learning Vanilla-Redux and React-Redux

## Project

### How to run?

```bash
# Clone repository
git clone https://github.com/dev-chloe/hangout-redux-with-nomad.git
cd hangout-redux-with-nomad

# Install node_modules
npm install

# Run application
npm start
```
